A méréseket az Ableton nevű zenekészítő programmal végeztem, a beépített spektrum plugin és a külső isotope insight 2 pluginnal, ami
a spektrogramot biztosította. Mindkettő egy-egy csatornához rendelhető a programon belül, a csatornán pedig könnyen, egymás után meg lehet
szólaltatni a felvett ének és gitárhangokat.

A következő mintákat gyűjtöttük:
hang típusa          C1 C2 C3 C4 C5 D6 E2 A1 C#1 C#2 D#1 D#4 H0 H1 G1 F#1 F1 C6 A#1 G#1 E1 D1

Basszusgitár         x  x  x  x                          x   x
Gitár                      x  x  x  x  x                                     x
Ének                  x x  x  x           x   x  x    x          x  x  x   x     x   x   x  x

Minden felvételen vizsgáltam a jellegzetes frekvenciák erősségét dbFS-ben, azok változását idő-frenvencia, valamint idő-jelszint függvényben.
Emellett egy hang jellemzőit az összes felvételt összevetve vizsgáltam.

C1_bass
3khz-ig terjedően jelennek meg hangok, amik exponenciálisan csengenek le frekvenciatartományban. A felvétel végén 1,5 khzig terjedően gyenge zaj található.
A hangerő a felvétel elején logaritmikusan emelkedik, majd az idővel arányosan lecseng. 
A C1 hang van legerősebben jelen, emellett -28 dbFSel C0, -24 dbFS-el G1, -28 dbFS-el G2, -34 dbFS-el E2, -40 dbFSel G2, -44 dbFSel A#2, -48 dbFSel C3, -50 dbFSel D3,
-49 dbFSel E3, -48 dbFSel F#3, -48 dbFSel G3, -49 dbFSel A3, -51 dbFSel A#3 -53 dbFSel B3, -53 dbFSel C4, -56 dbFSel D4 szólal meg. 
G3 magasságában szűrő rezonanciájának jellegzetes képe rajzolódik ki. A hangerő a felvétel elején logaritmikusan emelkedik, majd az idővel arányosan halkul, logaritmikusan cseng le.

C2_bass
4khzig jelennek meg hangok, lassabban csengenek le exponenciálisan, majd a felvétel végén 3khizg terjedő zajjal megszakadnak.
A frekvenciák jól láthatóan elkülönülnek a spektogramon. A hangerő a felvétel elején logaritmikusan emelkedik, majd az idővel arányosan halkul, logaritmikusan cseng le.
